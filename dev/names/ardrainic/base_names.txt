# ============================================================
# ARDRAINIC — Pool de noms de base (Ardrainien = standard)
# ============================================================
# Structure : mix formule (Attaque+Terminaison) + organiques + memorables
# Ratio : ~50% formule, ~25% organiques, ~15% communs, ~10% memorables
# Regle : 2-3 syllabes, max 5/terminaison, max 3/attaque, 25% organiques min
# Marqueurs exclusifs : -ain/-eval (max 5 chacun), Gal-/Thib-, voyelle au
# Les noms existants du mod (Adam, Louis, etc.) sont GARDES
# Ceux-ci s'AJOUTENT par-dessus
# ============================================================

# ============================================================
# REGLES DE TRANSFORMATION RESUMEES
# ============================================================
#
# Sittadellian : -val->-vard, -ain->-art, -en->-ent, -or->-ort,
#                -el->-eld, -in->-ird, consonnes durcies (fortifie/normand)
#
# Aullerican : -val->-vael, -ain->-aen, -or->-oer,
#              shift vocalique leger, touche germanique (frontiere flamande)
#
# Aloisoise : -val->-valen, -ain->-ainer, -or->-oren, -en->-ener,
#             ajout -en/-er (influence lorraine/germanique marquee)
#
# Caprainois : -val->-vel, -ain->-in, -or->-orel, -el->-el,
#              adoucissement, voyelles ouvertes (doux/ligerian)
#
# Pwasciards : tronque la derniere syllabe/consonne,
#              -val->-va, -ain->-a, -or->-o, -en->-e
#              noms courts, rudes (maritime/vendeen)
#
# Serenish : -val->-van, -ain->-an, -en->-on, -in->-on,
#            shift vocalique archaique e->o, i->o (montagnard/berrichon)
#
# Berenguer : -val->-wald, -ain->-ein, -or->-ort, -ic->-ich,
#             consonnes germaniques (hybride alsacien)
#
# Draquatin : -val->-vat, -ain->-oun, -en->-et, -or->-oun,
#             terminaisons gasconnes -at, -oun (dragon-gascon)
#
# Chevallois : -val->-vault, -ain->-ault, -en->-olt, -or->-ourt,
#              terminaisons lourdes -ault/-olt (forestier/bourguignon)
