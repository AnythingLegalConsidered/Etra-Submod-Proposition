# REGLES DE TRANSFORMATION RESUMEES
# ============================================================
#
# Fullhorner : +e intercalaire, a->ae, o->oe (archaique)
# Afhaemeisan : w->v parfois (legerement adouci)
# Lower Serenic : a->ae leger, -ld->-lt
# Middle Serenic : w->v, +influence ardrainic (-ain/-in)
# Sorlkol : a->o (montagnard, suisse)
# Naurmen : i->ee, a->aa, -wald->-woud, -mund->-mond, -hard->-haart,
#           -stein->-steen, -bern->-beer, -brand->-braand, -rik->-reek,
#           -kald->-koud, -dorn->-doorn, -varn->-vaan, -mar->-maar
# Stedeling : base neerlandaise (comme Naurmen) + influence ardrainic aux frontieres :
#             -woud->-voud parfois, -gard->-gain (ardrainic -ain), -kald->-kain
#             Resultat : hybride neerlando-francais reconnaissable
# Thaurfolk : -wald->-vald, g->j, patronymes -sen
# Bonwesforian : -mund->-mond (bas-allemand)
# Borrikan : terminal devoicing — d→t, b→p (East Germanic, Gothic-like)
# Tyelkian : a→e fronting/umlaut — wald→weld, gar→ger, mar→mer (High Germanic)
# Chesmeker : a→o darkening + -th terminal — wald→wolth, mund→month (Frontier Germanic)
# Erlfolk : PAS de transform — utilisent dialecte local + ~25% noms proto-elfiques germanises
# Tieflings : PAS de transform — utilisent south_torrentor_dialect
# Ratfolk (Knawven) : PAS de transform — utilisent north_torrentor_dialect
# Himmelskind : PAS de transform — utilisent south_torrentor_dialect
