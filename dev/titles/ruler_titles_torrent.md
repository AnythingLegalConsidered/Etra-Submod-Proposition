# Torrent — Ruler Titles (Monarchy)
# Inspiration: Germanic (Old High German / archaic)
# Base dialect: Heuhian (standard heartland)

## Design Rules
# - Titles are COMPOUND WORDS built from Torrent morphemes (same pool as names)
# - Pattern: recognizable root (Kaisar, Kunig, Herz, Graf) + Torrent morpheme
# - Each dialect has a SIGNATURE MORPHEME or STRUCTURAL CHANGE — not just phonetic shifts
# - Distant dialects (Thaurfolk, Tyelkian) can use completely different roots or structures
# - CRITICAL: Borrikan, Tyelkian, Chesmeker are ALL GERMANIC — NOT Slavic/Hungarian/Turkic
#   They are Germanic dialects with phonetic twists, not different language families.

## Morpheme Pool (from Torrent name ADN)
# Masculine: -mund (protector), -wald (rule), -vald (power), -gar (spear),
#            -varn (guardian), -hart (brave), -stein (stone), -brand (sword/fire),
#            -rik (ruler), -maar (glory), -bern (bear), -jar (spear, g→j Norse),
#            -mond (protector variant), -waeld (archaic rule)
# Feminine:  add -e to M morpheme (Germanic standard) OR -in for short titles (Kunigin)

## Base Titles (Heuhian = standard)

| Rank    | Ruler M     | Ruler F      | Etymology                              |
|---------|-------------|--------------|----------------------------------------|
| Empire  | Kaisarmund  | Kaisarmunde  | Kaisar (Caesar archaic) + mund (protector) |
| Kingdom | Kunig       | Kunigin      | Middle High German archaic for Koenig  |
| Duchy   | Herzgar     | Herzgarde    | Herz (from Herzog=army) + gar (spear)  |
| County  | Grafvarn    | Grafvarne    | Graf (judge) + varn (guardian)          |

## CRITICAL DESIGN DECISIONS
# - Borrikan is NOT Slavic. It is a Germanic dialect with VOICED consonants (K→G, S→Z, F→V).
#   The "buzzy" dialect. All roots stay Germanic.
# - Tyelkian is NOT Hungarian. It is a Germanic dialect with REVERSED compound order
#   and FRONTED vowels (u→ue, accents). The "melodic/exotic" dialect.
# - Chesmeker is NOT Turkic. It is a Germanic dialect with GUTTURAL aspiration
#   (K→Kh, H→Kh, G→Gh). The "frontier harsh" dialect.
# - These 3 cultures were previously designed with Slavic/Hungarian/Turkic inspirations
#   for their NAMES. This needs to be audited and corrected (see dialect_rules.txt).
