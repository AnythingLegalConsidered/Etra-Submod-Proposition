# Armonorican — Decisions

## Choix d'inspiration
- **Gallois/Breton** comme base, pas cornique ou gaelique — pourquoi : position peninsulaire nord-ouest = Bretagne, pas Irlande/Ecosse. Le mod original utilise deja des noms gallois (Angharad, Gwenllian, Dyddgu).
- **Fantasy-Celtic** (~40%) plutot que 100% reconstitution galloise — pourquoi : Etra est un monde fantasy, pas une reconstitution historique.

## Marqueurs exclusifs
- **Ll-, Rh-, Gw-** en attaque — pourquoi : sons les plus immediatement "gallois" pour un joueur. Aucune autre langue d'Etra ne les utilise.
- **-wyn, -wen, -ydd** en terminaison — pourquoi : terminaisons gallois classiques, absentes des autres langues.

## Structure des dialectes
- **Aggeuios = default** (pas Ebiam) — pourquoi : ruraux isoles preservent la forme ancienne. Parallele Basse-Bretagne (breton pur) vs Haute-Bretagne (gallo).
- **Ebiam = gallo-ardrainic** (Ll→L, Rh→R, Gw→Gui, w→v, dd→d) — pourquoi : simule l'influence francaise sur le breton en zone de contact.
- **Pas de 3eme dialecte** pour l'instant — pourquoi : seulement 2 cultures. Future culture "Gallo" possible.

## Approches rejetees
- **Filtre racial elfique** — rejete : humains purs.
- **Prefixes dynastiques ap_/mac_** — rejete pour dynasties : deja utilises par les lowborn.
- **Gw- → Gv-** pour Ebiam — rejete : imprononçable. Gw→Gu est le parallele historique correct.
- **Noms 100% composes** — rejete apres audit : trop mecanique. Ajout organiques/courts.
- **Mor-/Morv-** comme attaque — rejete apres audit cross-langue : friction ardrainic (Morvain).
- **Tal-** comme attaque — rejete apres audit cross-langue : friction ardrainic (Talise).

## Interactions avec d'autres langues
- **Ardrainic** : voisin principal. Influence = dialecte Ebiam. Aucun nom partage apres corrections.
- **Sordrenic/Torrent/Moran** : pas de frontiere, zero overlap.
- **Bran** : edge case — standalone celtique ici vs attaque "Braneval" en ardrainic. Contextes differents, accepte.
