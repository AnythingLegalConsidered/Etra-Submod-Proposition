# ============================================================
# DIALECT RULES
# ============================================================
#
# Aggeuios (rural, interior/west):
#   - Pure archaic Armonorican
#   - Conservative vowels, harder consonants
#   - No Ardrainic influence
#   - Example: Llewyn -> Llewyn (unchanged, base = archaic standard)
#
# Ebiam (urban, north-east border):
#   - Armonorican base + Ardrainic loanwords (Gallo-style)
#   - Softened endings: -wyn -> -vin, -ydd -> -id, -wedd -> -vedd
#   - Gw- -> Gu- (gallo-roman influence, NOT Gv-)
#   - Occasional Ardrainic endings: -ain, -el borrowed
#   - Example: Llewyn -> Llevin (softened double-L + -wyn > -vin)
#
