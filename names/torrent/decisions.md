# Torrentian — Decisions

## Inspiration
- Base : germanique (vieux haut-allemand, norrois, gothique)
- Fantasy : morphemes inventes qui SONNENT germanique (Skor-, Thar-, Drak-, -varn, -gust, -dorn, -kald, -morn)
- Sous-cultures non-germaniques : polonais (Borrikan), hongrois (Tyelkian), tatar (Chesmeker), danois (Thaurfolk), neerlandais (Naurmen/Stedeling)
- Pourquoi : le Holy Commonweal = coalition de 13 tribus, la diversite est VOULUE dans le lore

## Approches rejetees
- **Prenoms composes fantasy** (Sig-Volkmar) : rejete car les composes sont une convention de noblesse tardive, incoherent avec nos noms archaiques
- **Filtres raciaux** (elfes=liquide, ratfolk=sifflant, etc.) : remplace par le systeme substrat — les non-humains absorbent la culture locale
- **Ros- comme attaque feminine** : cree un conflit avec ardrainic qui utilise deja Ros- (Roseline, etc.) → remplace par Rost- (rouille)
- **Aldwin** : -win trop proche de -wyn armonorican → remplace par Aldven
- **Grimvaldorn, Drakenthor, Sigamund** (noms a 3 morphemes/allonges) : retires pour reduire la surutilisation de Grim-/Drak-/Sig-
- **Grimmorn** : retire par le user, remplace par Friedgust (redistribution)
- **Dialectes propres pour Tieflings/Ratfolk/Himmelskind** : abandonne au profit du systeme substrat

## Interactions avec d'autres langues
- **Ardrainic** : attaques partagees (Arn-, Ger-, Ald-, Brun-) differenciees par terminaisons. Middle Serenic et Stedeling montrent l'influence ardrainic (-ain/-vain).
- **Armonorican** : aucun partage direct. Aldven evite le conflit -win/-wyn.
- **Sordrenic** : aucun conflit. -orn torrent ≠ -orn sordrenic (contextes differents).
- **Proto-elfique** : Erlfolk = 25-30% noms proto-elfiques germanises (attaques Ael-, Fionn-, Ilma- + terminaisons torrent)
