# ============================================================
# REGLES DIALECTALES
# ============================================================

# SALT SORDRENIC (base / default)
# -----------------------------------------
# Standard. Noms tels quels.

# STONE SORDRENIC (montagnard, dur)
# -----------------------------------------
# Voyelles : e→o (fermeture)
# Terminaisons M : -enc→-orc, -tz→-rtz, -esc→-osc, -enzo→-orzo
# Terminaisons F : -enca→-orca, -ella→-olla, -onda→-orda, -ina→-orna
# Consonnes : durcies (s→ss, c→k)
# Style : plus court, tronque, rude
#
# Exemples :
# Esclarenc → Esclarorc | Lorenca → Lororca
# Castenzo → Castorzo  | Fiorella → Fiorolla
# Arnotz → Arnortz     | Garsanda → Garsorda

# LUTIN (culture pont — sordrenic + ardrainic + proto-elfique)
# -----------------------------------------
# Les Lutin vivent en foret centrale ENTOUREES par les Ardrainic
# mais parlent sordrenic. Pool = mix triple :
#
# ~35-40% Sordrenic (leur langue) : Esclarenc, Castenzo, Fiorella
# ~30-35% Ardrainic (leurs voisins) : Galenor, Thibeval, Belisore
# ~15% Proto-elfique filtre Sordrenic : Aelvintz, Niamella, Suraniesc
# ~15% Proto-elfique filtre Ardrainic : Aelvain, Nimueline, Suraneval
#
# Filtre proto-elfique → Sordrenic : -dh→-tz, -ael→-ella, -iel→-iesc
# Filtre proto-elfique → Ardrainic : -dh→-ain, -ael→-eline, -iel→-eval
#
# Exemples de filtrage double :
# | Proto pur    | Filtre Sordrenic | Filtre Ardrainic |
# |-------------|-----------------|------------------|
# | Aelvindh    | Aelvintz        | Aelvain          |
# | Niamhael    | Niamella        | Nimueline        |
# | Suraniel    | Suraniesc       | Suraneval        |
# | Tuathiel    | Toatienc        | Toathelin        |
# | Fionneth    | Fionesc         | Fionneval        |
#
# Les noms sordrenic du pool Lutin recoivent aussi le twist
# dialectal lutin (adoucissement, insertion "i" liquide) :
# Esclarenc → Esclarienc | Lorenca → Lorienca
