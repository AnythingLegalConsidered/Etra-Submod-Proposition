## Topographie (topographie.png)

- **Vert clair** = plaines basses fertiles
- **Kaki/olive** = collines, plateaux
- **Marron fonce** = montagnes (sommets enneiges en blanc)
- **Bleu** = mer/cotes

**Observations cles :**
- Grand fleuve MASSIF au nord-ouest (frontiere Armonor/Ardraine)
- Chaine montagneuse centrale est-ouest = frontiere naturelle Ardraine/Sordren
- Montagnes massives a l'est = frontiere Torrent/Hravevi
- Peninsule sud (Moran) = montagneuse avec vallees (type Apennins)

## Foret (forest.png)

- **Vert clair** = boisements legers
- **Vert moyen** = forets denses
- **Vert fonce** = forets tres denses/anciennes

**Observations cles :**
- Nord ardrainic = massivement boise → theme "Broceliande"
- Centre-ouest (zone Chevallois) = bloc de foret TRES dense → "Heartwoods"
- Cote ouest = moins boise, degage
- Sud-ouest (draquatin/sordrenic) = clairseme, sec
- Peninsule moran = patchwork foret/degage

## Langues dominantes (language.png)

6 zones nettes :
1. **Ardrainic** (violet) = tout l'ouest, plus grande zone
2. **Torrentian** (gris) = nord-est + est
3. **Sordrenic** (bleu clair) = bande sud-ouest
4. **Freeholdan** (rouge) = zone tampon centre
5. **Moran** (or) = peninsule sud
6. **Armonoric** (gris clair) = petit coin nord-ouest
7. **Hravevian** (rose) = enclave est

Frontiere Ardrainic/Sordrenic = pile sur la chaine de montagnes centrales.
Freeholdan au carrefour de tout = logique pour une lingua franca.

## Observations transversales

1. **Les elfes gardent les frontieres** : Lutin (foret centre ardrainic) et Erlfolk (foret frontiere ardrainic/torrent)
2. **Les ratfolk dans les interstices** : zones hachurees entre grandes cultures, opportunistes urbains
3. **Les tieflings au coeur du torrent** : emerges en 1078, relativement recents
4. **Montagnes = frontieres linguistiques** : chaine centrale separe ardrainic/sordrenic, montagnes est separent torrent/hravevi
5. **Le grand fleuve nord-ouest** : artere commerciale, frontiere naturelle Armonor/Ardraine
6. **La Grande Migration (1070+)** : explique le melange aux frontieres
7. **Le Torrent = coalition de 13 tribus** : explique la diversite enorme des sous-cultures
8. **Religion** : Eternalist partout sauf Moran (Imperial Creeds) → pas d'impact direct sur les noms mais sur l'ambiance
